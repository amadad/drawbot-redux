# DrawBot Design Library

A comprehensive educational resource for learning design programming with DrawBot, inspired by classical design principles and modern computational approaches.

## Overview

This library provides a structured learning path from basic programming concepts to advanced design techniques, all implemented in DrawBot. Whether you're a designer learning to code or a programmer exploring visual design, you'll find resources tailored to your journey.

## ğŸš€ Getting Started

### Prerequisites
- Python 3.8+
- DrawBot (installed via pip or DrawBot.app)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/drawbot-design-library.git
cd drawbot-design-library
```

2. Install DrawBot:
```bash
pip install drawbot-skia
```

### Running Examples

From the project root:
```bash
# Run a basic example
python drawbot-design-library/basics/01_first_steps.py

# Run a specific tutorial
python drawbot-design-library/tutorials/beginner/shapes_and_color.py
```

## ğŸ“– Learning Paths

### For Complete Beginners
1. Start with `basics/` - Learn fundamental concepts
2. Move to `tutorials/beginner/` - Build your first projects

2. Jump to `tutorials/intermediate/` - Advanced techniques

### For Experienced Programmers New to Design
1. Start with `foundations/` - Learn design principles
2. Explore `historical_studies/` - Understand design context
3. Try `interactive/` - Create dynamic designs
4. Challenge yourself with `tutorials/advanced/`
